# CodeBuddy: Competitive Practice Tracker

Welcome to **CodeBuddy**! This is a desktop application I built to help competitive programmers track their problem-solving progress, analyze their activity, and stay motivated. I developed this project with the help of an AI coding assistant, which guided me through design, implementation, and debugging.

## 🚀 What is CodeBuddy?
CodeBuddy is a JavaFX-based desktop app for tracking competitive programming problems you solve across different platforms (like LeetCode, CodeChef, etc.). It lets you log problems, view analytics (streaks, platform/difficulty breakdowns, monthly stats), and manage your practice history—all in a user-friendly interface.

## ✨ Features
- **Add/Edit/Delete Problems:** Log problems with details (name, platform, difficulty, time, notes, link).
- **Analytics Dashboard:** Visualize your progress with streaks, pie charts, and bar charts.
- **Recent Activity Table:** See your latest problems, with customizable row count.
- **User Authentication:** Secure login and user management.
- **Dark/Light/High-Contrast Themes:** Switch themes for accessibility.
- **Export/Import:** Save or load your data as CSV or JSON.

I started this project to organize my competitive programming journey. With the help of an AI assistant, I:

## 📦 Tech Stack
- **Java 17**
- **JavaFX 21** (UI)
- **SQLite** (local database)
- **Maven** (build & dependency management)
- **JUnit 5** (testing)

## 🖥️ How to Run
1. **Clone the repo:**
   ```sh
git clone https://github.com/learnerview/codebuddy.git
cd codebuddy
```
2. **Build the project:**
   ```sh
mvn clean install
```
3. **Run the app:**
   ```sh
mvn javafx:run
```

## 📋 Usage
- **Login/Register:** Start by logging in or creating a new user.
- **Add Problems:** Use the UI to add new problems as you solve them.
- **View Analytics:** Click the Analytics menu to see your stats and streaks.
- **Change Theme:** Use the theme toggle for dark/light/high-contrast modes.
- **Export/Import:** Use the File menu to backup or restore your data.
